しほしけコンバーター（β版）

司法試験論文式の試験問題・出題の趣旨・採点実感を、法務省ウェブの公表PDFからテキストに変換します

短答式（司法試験・予備試験）の問題・正答PDFを年度別に保存したい場合は しほしけ短答ダウンローダー をどうぞ。

更新情報

    ヘルプ（よくある質問）

    このツールは何ですか？

    司法試験論文式の「試験問題」「出題の趣旨」「採点実感」を、法務省ウェブサイトで公表されているPDFから自動で抽出してテキスト化する、法務省非公式の個人開発ツールです。処理はすべてブラウザ内で行われ、登録やデータ収集はありません。

    基本的な使い方は？

    年度・科目・種類・出力形式を選んで「変換実行」を押すと、結果欄にテキストが表示されます。「結果をコピー」でクリップボードへ、「.txt 保存」でファイルとして保存できます。

    対応している年度・科目は？

    平成22年から最新年度までの論文式です。科目は基本7科目（憲法・行政法・民法・商法・民訴・刑法・刑訴）と、選択科目のうち経済法・労働法・倒産法（各第１問・第２問）に対応しています。

    予備試験にも対応していますか？

    画面上部の「試験」で予備試験に切り替えると、論文式の試験問題と出題の趣旨を科目別の原典PDFで保存できます（平成23年〜最新年度）。予備試験は科目をまとめたPDF（憲法・行政法 など）で公表され、出題の趣旨も全科目を1つにまとめたPDFですが、本ツールが選択した科目だけを自動で切り出します（一部の年度は趣旨が画像PDFのため切り出せず全体を保存します）。採点実感はありません。これらの事情からテキスト変換は行わず、出典フッター・見出し付きのPDF保存に対応しています。短答式は しほしけ短答ダウンローダー が司法試験・予備試験の両方に対応しています。

    新しい年度はいつ使えるようになりますか？

    毎週月曜の朝に法務省ウェブサイトを自動巡回しています。試験問題（例年8月頃）、出題の趣旨・採点実感（例年9〜11月頃）が法務省ウェブに掲載されると、1週間以内に自動で追加され、「更新情報」欄でお知らせします。

    出力形式の「ノーマル」と「Scrapbox記法」の違いは？

    ノーマルは装飾のないプレーンテキストです。Scrapbox記法を選ぶと、見出しが [* ]・[** ] で装飾され、#司法試験 などのタグ行が付きます。どちらも冒頭に出典（法務省の原典PDFのURL）が入ります。

    「原典PDF保存」は何をしますか？

    選択中の年度・科目・種類について、法務省の原典PDFから該当するページだけを抜き出したPDFを保存します。各ページの右下にファイル名、左下に出典と、試験問題・出題の趣旨・採点実感それぞれの原典PDFへのクリック可能なリンクを印字します。変換実行をしていなくても、押せば自動で取得・処理されます。

    「一式zip保存」は何をしますか？

    選択中の年度・科目について、試験問題・出題の趣旨・採点実感の3点の抜粋PDFを「[年度]司法試験[科目]一式」フォルダにまとめ、zipで保存します。処理済みの種類は再利用されるため、変換実行のあとに押すと速く完了します。未掲載の種類がある年度では、取得できた分だけが入ります。

    「LLM用に保存」は何をしますか？

    選択中の年度・科目について、試験問題・出題の趣旨・採点実感を1つのMarkdownファイルにまとめて保存します。冒頭に「これは司法試験論文式の過去問である」というメタ情報と出典（原典PDFのURL）が入るので、ChatGPTやClaudeなどのLLMにそのまま貼り付け・添付して、答案の添削や解説を依頼するのに使えます。

    選択科目の採点実感で、第１問と第２問の内容が同じになります

    原典の採点実感が問ごとに分かれていない年度・科目（選択科目や平成期の多く）では、科目全体（第１問・第２問の両方を含む）を出力します。原典の構造上の仕様です。

    抜粋PDFの先頭ページに前の問題の終わりが入っています

    抜粋はページ単位で行うため、同じページ内で前の問題が終わり次の問題が始まっている場合、その部分が含まれます。

    出力が原典と違う気がします

    PDF解析による自動変換のため、改行位置や段落の区切りなどに誤りが生じることがあります。学習には原典PDFをあわせてご確認ください。おかしな箇所があれば GitHub の Issue からお知らせください。

    著作権は大丈夫ですか？学内などで共有してもよいですか？

    法務省ウェブサイトのコンテンツには「公共データ利用規約（PDL1.0）」が適用されており、出典の記載と加工した旨の表示を条件に、複製・共有・加工が認められています。本ツールの出力には出典・加工表示が自動で付くので、そのまま保持して共有してください。本ツール自体の共有も自由です。
